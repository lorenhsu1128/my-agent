根據我的指示建立一個新的 skill。

如果我沒有指定 skill 的主題，請問我要建立什麼 skill。

建立步驟：
1. 確認 skill 名稱和主題
2. 在 `.claude/skills/` 下建立新目錄：`.claude/skills/{skill-名稱}/`
3. 在該目錄下建立 `SKILL.md`，遵循以下格式：

```markdown
# [Skill 標題]

## 說明
[這個 skill 是什麼、何時應該載入]

## 工具集
[需要哪些工具：file, terminal, 等]

## [主要內容]
[具體的知識、步驟、模式、注意事項]

## 參考
[相關檔案、文件連結]
```

4. 內容要求：
   - 使用繁體中文
   - 包含具體的程式碼範例或指令（如適用）
   - 記錄已知的陷阱和注意事項
   - 標註與其他 skill 的關聯（如有）

5. 建立完成後：
   - 在 TODO.md 的 session 日誌中記錄新建了哪個 skill
   - 告訴我 skill 已建立，以及它的用途摘要
